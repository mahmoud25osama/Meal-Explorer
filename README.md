# Meal Explorer

**Meal Explorer** is an interactive web project that showcases delicious meals from the [TheMealDB](https://www.themealdb.com/) API.  
You can choose a category (like Beef, Chicken, Dessert, etc.), view the related meals, and click on one to see full details â€” including ingredients, cooking instructions, and even a YouTube tutorial video (if available).
